## What is DentalConnect?

DentalConnect is a complete clinic management system developed by Sakari, designed specifically for dental practices. It provides a secure, multi-tenant environment where each clinic operates in complete data isolation, with a full suite of operational tools and a SaaS subscription layer for platform access.

## Platform architecture

DentalConnect is a **multi-tenant** Laravel application. Each dental clinic registers as a tenant and all data — patients, appointments, staff, and billing — is scoped to that clinic's `id_clinica`. There is no shared data between clinics.

```
Clinic (id_clinica)
├── Users (roles: administrador, doctor, recepcionista, paciente)
├── Patients → Appointments → Treatments → Odontogram
├── Inventory & Cash Register
├── Advertising (Premium+ only)
└── SaaS Subscription (Basic / Premium / Ultra)
```

## Roles

| Role | Description |
|------|-------------|
| `administrador` | Full access to clinic settings, staff, and the global admin panel |
| `doctor` | Access to patient records, appointments, treatments, and odontogram |
| `recepcionista` | Can manage appointments and view patient lists |
| `paciente` | Patient-facing access via token-based login |

## Tech stack

- **Framework**: Laravel 12 (PHP 8.2+)
- **Frontend**: Blade templates with Vite asset pipeline
- **Database**: MySQL (Eloquent ORM)
- **Payments**: Stripe Checkout + Webhooks
- **PDF generation**: barryvdh/laravel-dompdf
- **Hosting**: Procfile-based deployment (Railway/Render compatible)
